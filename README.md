# data1030project
For this project, I utilized a simulated dataset for the purposes of creating a binary classification model on the prediction of credit card fraud. I chose to use a simulated dataset because actual sources on credit card fraud contain classified features. For the purposes of the assignment, I found that it would be more interesting to use a simulated dataset with real features I could investigate rather than redacted ones. 

This project was done in Python 3 and utilized the pandas, numpy, and sci-kit learn libraries.
