# data1030project
For this project, I utilized a simulated dataset for the purposes of creating a binary classification model on the prediction of credit card fraud. I chose to use a simulated dataset because actual sources on credit card fraud contain classified features. For the purposes of the assignment, I found that it would be more interesting to use a simulated dataset with real features I could investigate rather than redacted ones. I developed Random Forest Classifier with an accuracy of 99.9% and just three false predictions in the confusion matrix.  

This project was done in Python 3 and utilized the pandas, numpy, and sci-kit learn libraries. It was completed entirely in a Jupyter Notebook. 

The dataset is too large to upload to GitHub, but it can be publicly accessed at https://www.kaggle.com/datasets/dhanushnarayananr/credit-card-fraud
